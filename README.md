Psychiatric Tool by Dr Benjamin Murrie

[Ben's Amazing ADHD Assessment Generator](adhd4.html)
This has everything you need. An interview template to follow with the key things plus DSM 5 TR criteria. Psychoeducatoin. Titration algorithm. Titration and follow up plan generator. Creates a nice adhd report, certificate, script text for e scripts, creates all the letters they might need. Has prescriber guidance per state. Everything. Perfect for beginners and experienced psychiatrists alike. 

[Old ADHD assessment generator](adhd3.html)
This was the first one I made. Subtle differences and not as good. 

[ADHD Handout Generator](ADHD-Patient-Guide-Generator-30.html)
This is basically already in the above apps. It's a web app that you can use to provide psychoeducation to your patient, and create a titration schedule for them. It generates a nice handout to print or send to your patients. Plus generates the plain text to put in your eScript. You can choose how much info to add to the hand out. Some patients need more and some need less. It also includes non-pharm options. Most of this has been put into the amove ADHD assessment generator.
